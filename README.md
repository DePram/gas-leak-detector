Simple IoT for Gas Leak Detector which is integrated with Platform Blynk
----------------------------------------------------------------------------
Component Use : 
- one Breadboard
- one ESP32-WROOM-32D
- one MQ-6 Senor
- one Buzzer 5V
- one LED 5mm
- one Solenoid Valve 12V
- one Step Up converter
- one Relay 1 channel
- one resistor 150-220 ohm
- Cable Jumper

Software Use :
- Arduino IDE
- Blynk IoT app
- Internet
- Latest Blynk Library
